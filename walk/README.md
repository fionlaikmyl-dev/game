# 2D Walking Game for Primary Students

This project is a simple 2D walking game designed for primary students to learn directions such as 'walk across', 'turn left', 'turn right', and 'walk along the map'. The game features a colorful map with places like a department store and a convenience store. Built using HTML5, CSS, and JavaScript for easy access and play in any modern browser.

## Features
- 2D map with walkable paths
- Simple controls for walking and turning
- Colorful, child-friendly graphics
- Landmarks: department store, convenience store, and more
- Educational prompts to guide students

## How to Play
1. Open `index.html` in your browser.
2. Use on-screen buttons or arrow keys to move your character.
3. Follow the prompts to practice directions.

## Folder Structure
- `index.html` - Main game page
- `style.css` - Game styles
- `game.js` - Game logic
- `assets/` - (Optional) Images for map and places

## Customization
You can add more places or change the map by editing `game.js` and `index.html`.
